# Changelog

All notable changes to EcotaleCoins will be documented in this file.

## [1.0.0] - 2026-01-18

### First Release

Physical currency and banking system for Hytale.

### Features

#### Physical Coins
- **6 coin denominations**: COPPER, IRON, COBALT, GOLD, MITHRIL, ADAMANTITE
- **World entity drops** - Coins spawn as collectible items
- **Optimal breakdown** - Automatically uses highest denominations
- **Gravity and pickup** - Coins fall and players can walk over them to collect

#### Bank System
- **Bank GUI** - Full banking interface with NoesisGUI
- **Deposit/Withdraw** - Convert between bank balance and physical coins
- **Exchange** - Convert between coin types
- **Consolidate** - Combine small coins into larger ones
- **Balance display** - Shows bank balance in HUD

#### Commands
- `/bank` - Opens bank GUI
- `/bank deposit <amount>` - Deposit coins
- `/bank withdraw <amount>` - Withdraw coins

#### API
- `PhysicalCoinsProvider` interface for other plugins
- `dropCoinsAtEntity()` - Drop coins at entity position
- `dropCoinsAtPosition()` - Drop coins at world coordinates
- Secure transaction handling with rollback on failure

### Permissions
- `ecotale.ecotalecoins.command.bank` - Required for /bank access
  - Note: This is auto-generated by Hytale, must be granted via LuckPerms

### Technical
- Integration with Ecotale API
- Used by EcotaleJobs for mob kill and mining rewards
- Thread-safe coin operations
